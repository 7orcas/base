

# LoginOptions
LoginOptions is a controller that returns the options for the login process. 
It is used to configure laguage choice, organisation choice, etc.

The URL must be called with '/LoginNr' appended.
This is used to get the options (store in cntrl.loginoption table).



# Development Only
- To auto login set "AUTO": "true" in the Properties/launchSettings.json file. 
- Change "launchUrl": "1" in Properties/lauchSettings.json to open different login options.
